# Crimson Watch - VS Code Project
A starter VS Code project skeleton for the **Crimson Watch** (Online Crime Reporting & Management System).

Structure:
- backend/         -> Node + Express API
- frontend/        -> Vite + React UI
- .vscode/         -> launch + debug settings for running frontend & backend
- .gitignore

How to use
1. Install Node.js (v16+ recommended).
2. From project root: `npm install` then `npm run dev` to start both frontend and backend concurrently.
3. Backend runs on http://localhost:5000, Frontend on http://localhost:5173
